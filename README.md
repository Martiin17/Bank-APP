# BANK APP

Este proyecto consiste en el desarrollo de un sistema de simulación bancaria orientado a objetos, diseñado con el propósito de validar modelos de interacción entre entidades financieras, sucursales y clientes. El enfoque principal de la investigación técnica se centró en la creación de prototipos funcionales y la implementación de un marco de pruebas basado en BDD (Behavior-Driven Development) utilizando el lenguaje Gherkin, asegurando que los requerimientos del dominio se traduzcan fielmente en lógica ejecutable.
